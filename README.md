# 🎌 Guess The Flag Game

An interactive and fun web-based game that challenges users to identify world flags. Test and improve your knowledge of flags from around the world!

## 🎮 How to Play

1. A flag image is displayed on the screen
2. Choose the correct country from **3 multiple choice options**
3. The game includes **15 flags** from around the world
4. Try to get a perfect score!

## 🛠️ Tech Stack

- **HTML5**
- **CSS3**
- **JavaScript**

## 🚀 How to Run

1. Clone the repository:
```bash
git clone https://github.com/husseinaljaber/Advanced-Programming-Techniques.git
```
2. Open `index.html` in your browser
3. Start playing!
